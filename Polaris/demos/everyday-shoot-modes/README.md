# What To Look For

Do not only look for a clever answer.

Look for whether the engine makes the problem clearer.

| Check | Question |
|---|---|
| clarity | Is the problem easier to see? |
| route | Are there real options? |
| hidden tension | Did the output reveal a hidden pressure? |
| counterexample | Did it show where the answer may fail? |
| next action | Can you do something now? |

# FAQ

## Is this only for serious problems?

No.

A small problem can still reveal structure.

## Is this therapy, legal advice, medical advice, or financial advice?

No.

Use it as a thinking tool.  
For professional matters, use professionals.

## Can it read someone’s mind?

No.

It can analyze visible signals and possible interpretations.  
It should not pretend to know a hidden truth with certainty.

## Why include memes and cats?

Because the point is generality.

If an engine can structure a world-class problem and a tiny absurd problem, it is not just a narrow tool.

## What is the fastest way to try it?

Use:

```text
shoot + [your problem]
```
